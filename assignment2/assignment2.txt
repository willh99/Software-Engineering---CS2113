	In order to prove that this program functions correctly, various conditions were applied to the GUI in order to assess its functionality. These included changing the size of the landscape, changing the fill density, changing the fraction of type 1 trees, and changing the positions of the ignition points. See the commented out code in main of Assign2.java as well as the pictures in this directory. 
	There are also various print statements throught the code that were used for debugging which also served to demonstrate the fuctionality once the proper values were printed.
